Colors

All color tokens are CSS variables in tokens.css. Each ramp has 50–900 steps. Semantic roles map to specific ramps.

Roles: --color-primary (lime, action) · --color-secondary (forest, contrast surfaces) · --color-danger (coral, errors/destructive) · --color-neutral-* (text, borders, surfaces)
